## 📚 学到的东西

### 1. 工作流设计哲学
- **流式 > 批处理**: Checkpoint/Recap比start/end-working更适合连续工作
- **文件 > 对话**: 对话会compress，文件不会
- **三层记忆**: auto memory + 日志 + 快照，缺一不可

### 2. GitHub自动化策略
- **Prophet自己营销自己**: 元循环，AI驱动AI
- **Actions + Git = 活跃度**: 持续commit展示进化
- **真实数据 > Demo**: 实际运行的stats更有说服力
